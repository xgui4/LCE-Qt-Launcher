Version Nightly

[!NOTE]
Cette branche n'est pas stable et des modifications y sont apportées presque quotidiennement ; elle peut donc parfois dysfonctionner. De plus, macOS n'est pas disponible dans la version Nightly en raison des restrictions d'Apple et du fait que je ne possède pas de Mac.

Sur cette page GitHub Release, vous trouverez les versions Nightly, générées automatiquement via GitHub Actions lors de modifications dans la branche dev

Via Git

Vous pouvez également télécharger ce dépôt avec la commande : git clone https://github.com/xgui4/lce-qt-launcher.git puis compiler (voir docs/en/COMPILING.md pour plus d'informations) manuellement ou exécuter dans un environnement virtuel (.venv) (voir docs/en/RUNNING.md pour plus d'informations).
